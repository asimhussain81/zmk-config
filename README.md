# ZMK config

This repository now uses the official Redox shield keymap in `config/redox.keymap` and a matching build matrix in `build.yaml` for `nice!nano`.

To keep refining the right-hand layer in the browser, open the keymap editor at:
https://nickcoutsos.github.io/keymap-editor/

Then load this repository as the ZMK config source and adjust the right-hand N/M and navigation positions there.
